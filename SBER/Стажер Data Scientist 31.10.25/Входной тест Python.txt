1) Какие из утверждений верны? (Выберите все верные варианты ответа.)
1. При открытии файла на запись, если файл не существует, возникнет ошибка
2. При открытии файла на запись, если файл не существует, создастся новый файл
3. При открытии файла на запись, если файл существует, он перетрется новым

5) Дан список строк  
codes = ['Alpha', 'Bravo', 'Charlie', 'Delta', 'Echo']  
Какое из представленных выражений выведет на экран все коды, каждый на своей строке соответственно?
1. print(codes.concatenate("\n"))
2. print(codes.join("%s\n", codes))
3. print("\n".join(codes))
